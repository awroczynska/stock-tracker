# Stock Price Tracker — Python

A command-line stock price tracker built in Python using the Alpha Vantage API.

## Features
- Look up real-time stock prices by symbol
- Displays price, change, high, low and volume
- Supports any stock listed on major exchanges (e.g. AAPL, TSLA, MSFT)

## How to run
1. Install requests: pip3 install requests
2. Run: python3 stock_tracker.py
3. Enter a stock symbol when prompted (e.g. IBM, AAPL, TSLA)

## API
Uses the Alpha Vantage API (alphavantage.co). Replace API_KEY = "demo" with your own free key for full access.

## Concepts used
REST API · JSON parsing · Python functions · Loops · User input
